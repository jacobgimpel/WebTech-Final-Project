# Web Technologies Final Project 

The final project for Web Technologies is to design and build a portfolio to showcase 
current and future coding projects. 
This portfolio will display and include the elements covered throughout the cluster,
including the use of:
- HTML
- CSS Styling
- Javascript
- Tailwind
